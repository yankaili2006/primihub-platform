# PrimiHub测试框架使用说明

## 概述

已创建完整的测试框架来测试联邦学习功能。测试脚本位于：
```
primihub-test-framework/tests/suites/03_project_task/test_federated_learning_flow.py
```

## 测试流程

该脚本测试完整的联邦学习流程：

1. **用户登录** - 使用admin账户登录系统
2. **获取机构列表** - 获取可用的合作机构
3. **创建数据资源** - 模拟两方各创建一个数据资源
4. **创建联邦学习项目** - 创建包含两个合作方的项目
5. **查看项目详情** - 验证项目创建成功
6. **查询任务列表** - 查看相关的任务

## 快速修复方案

### 方案1：使用已登录的token

如果你已经有一个有效的token，可以直接设置到客户端：

```python
from api_client import PrimiHubAPIClient

client = PrimiHubAPIClient("http://172.20.0.12:8080")

# 直接设置token（从浏览器开发者工具获取）
client.token = "your_token_here"
client.user_id = 1
client.user_name = "admin"

# 然后就可以调用其他API了
organs = client.get_organ_list()
print(organs)
```

### 方案2：完善登录功能

需要在 `api_client.py` 中添加：

```python
import requests
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_v1_5
import base64

def login(self, username: str, password: str) -> Dict[str, Any]:
    """用户登录（带RSA加密）"""

    # 1. 获取公钥
    response = self._make_request("GET", "/sys/common/getValidatePublicKey")
    if response.get('code') != 0:
        raise Exception("获取公钥失败")

    result = response.get('result', {})
    public_key = result.get('publicKey')
    public_key_name = result.get('publicKeyName')

    # 2. 使用RSA加密密码
    rsa_key = RSA.importKey(public_key)
    cipher = PKCS1_v1_5.new(rsa_key)
    encrypted = cipher.encrypt(password.encode('utf-8'))
    encrypted_password = base64.b64encode(encrypted).decode('utf-8')

    # 3. 发送登录请求
    login_data = {
        'userAccount': username,
        'userPassword': encrypted_password,
        'validateKeyName': public_key_name
    }

    response = self._make_request("POST", "/sys/user/login", data=login_data)

    # 保存认证信息
    if response.get('code') == 0:
        result = response['result']
        self.token = result.get('token')
        self.user_id = result.get('userId')
        self.user_name = result.get('userName')

    return response
```

同时修改 `_make_request` 方法，使POST请求使用form-urlencoded格式：

```python
# 在_make_request方法中
if method.upper() == 'POST':
    # 使用form-urlencoded格式
    import urllib.parse
    if data is not None:

        # 转换为form-urlencoded格式

```

## 测试数据说明

测试脚本会创建以下测试数据：

### 数据资源1（机构1）
- 名称：联邦学习测试数据_机构1_{timestamp}
- 类型：CSV
- 字段：id, age, income, label
- 行数：1000

### 数据资源2（机构2）
- 名称：联邦学习测试数据_机构2_{timestamp}
- 类型：CSV
- 字段：id, education, score, feature
- 行数：1000

### 联邦学习项目
- 名称：联邦学习测试项目_{timestamp}
- 类型：联邦学习
- 模式：横向联邦学习
- 合作方：2个机构

## 测试报告

测试完成后会自动生成三种格式的报告：

1. **JSON报告** - `federated_learning_flow_{timestamp}.json`
   - 结构化数据，便于程序解析

2. **HTML报告** - `federated_learning_flow_{timestamp}.html`
   - 可视化报告，可在浏览器中查看
   - 包含通过率、测试详情、错误信息等

3. **Markdown报告** - `federated_learning_flow_{timestamp}.md`
   - 文本格式，便于分享和查看

报告保存在 `primihub-test-framework/tests/reports/` 目录

## API客户端功能

`api_client.py` 提供了完整的API封装：

### 认证相关
- `login(username, password)` - 用户登录
- `logout()` - 用户登出

### 用户管理
- `create_user(user_data)` - 创建用户
- `get_user_list(page, page_size)` - 获取用户列表
- `freeze_user(user_id)` - 冻结用户
- `unfreeze_user(user_id)` - 解冻用户

### 机构管理
- `create_organ(organ_data)` - 创建机构
- `get_organ_list()` - 获取机构列表

### 资源管理
- `create_resource(resource_data)` - 创建资源
- `get_resource_list(page, page_size)` - 获取资源列表

### 项目管理
- `create_project(project_data)` - 创建项目
- `get_project_list(page, page_size)` - 获取项目列表
- `get_project_detail(project_id)` - 获取项目详情

### 任务管理
- `get_task_list(page, page_size)` - 获取任务列表
- `get_task_detail(task_id)` - 获取任务详情

### 隐私计算
- `create_psi_task(psi_data)` - 创建PSI任务
- `get_psi_task_list(page, page_size)` - 获取PSI任务列表
- `create_pir_task(pir_data)` - 创建PIR任务
- `get_pir_task_list(page, page_size)` - 获取PIR任务列表

### 系统功能
- `create_whitelist(whitelist_data)` - 创建白名单
- `get_whitelist_list(page, page_size)` - 获取白名单列表
- `health_check()` - 健康检查
