🔑 第一步：获取Token

【获取node0的token】
1. 浏览器访问: http://172.20.0.12:8080
2. 使用admin账户登录
3. 按F12 → Network → 找到任意请求 → Headers → token
4. 复制完整的token值

【获取node1的token】
1. 浏览器访问: http://172.20.0.2:8080
2. 使用admin账户登录
3. 按F12 → Network → 找到任意请求 → Headers → token
4. 复制完整的token值

⚙️  第二步：配置脚本

cd /home/primihub/primihub-platform/primihub-test-framework/tests

vim add_resources_to_nodes.py

# 修改以下行（约第14-19行）：
NODE0_URL = "http://172.20.0.12:8080"
NODE0_TOKEN = "你的node0_token"  # ← 填入node0的token

NODE1_URL = "http://172.20.0.2:8080"
NODE1_TOKEN = "你的node1_token"  # ← 填入node1的token

# 保存退出：按Esc，然后输入 :wq
