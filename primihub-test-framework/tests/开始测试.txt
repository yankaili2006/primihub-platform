📝 第一步：获取Token

1. 在浏览器中打开PrimiHub并登录
2. 按 F12 打开开发者工具
3. 切换到 Network 标签
4. 刷新页面
5. 点击任意请求，查看 Headers 中的 token 字段
6. 复制完整的token值

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 第二步：运行测试

执行以下命令：

cd /home/primihub/primihub-platform/primihub-test-framework/tests

vim suites/03_project_task/test_with_token.py
# 修改第20行：USER_TOKEN = "你复制的token"
# 保存退出：按 Esc 然后输入 :wq

python3 suites/03_project_task/test_with_token.py
