## 💡 关键技术细节

### 字段类型映射
```
前端/API     → Java类型 → CSV类型
"String"     → String   → text
"Integer"    → Integer  → numeric
"Double"     → Double   → decimal
```

### resourceFusionId生成逻辑
```java
// OrganConfiguration.java
organId = "00000000000000000000000000000001"
shortCode = organId.substring(24, 36)  // "000000000001"
fusionId = shortCode + "-" + UUID.randomUUID()
// 例如: "000000000001-a1b2c3d4-..."
```

```protobuf
}
```
