1. your_system_tool.py ⭐⭐⭐⭐⭐ 推荐!
   用途: 管理你的系统资源
   命令:
     python3 your_system_tool.py list   # 查看资源列表
     python3 your_system_tool.py test   # 查看创建建议
     python3 your_system_tool.py help   # 显示帮助

针对标准节点 (172.20.0.x:8080):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. create_resources_interactive.py
   用途: 交互式创建资源 (引导式)
   命令:
     python3 create_resources_interactive.py

3. add_resources_to_nodes.py
   用途: 批量创建资源 (配置后运行)
   命令:
     vim add_resources_to_nodes.py  # 先配置token
     python3 add_resources_to_nodes.py

4. get_token.py
   用途: Token管理工具
   命令:
     python3 get_token.py test <url> <token>  # 测试token
     python3 get_token.py save node0 <token>  # 保存token
     python3 get_token.py load node0          # 加载token
     python3 get_token.py help                # 帮助

5. api_example.py
   用途: API功能演示
   命令:
     python3 api_example.py           # 运行演示
     python3 api_example.py --help    # 查看所有API
