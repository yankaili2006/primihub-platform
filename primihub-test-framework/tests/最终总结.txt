  1. 浏览器访问:
     http://100.64.0.23:30811

  2. 登录系统

  3. 进入"数据资源管理"或类似菜单

  4. 点击"创建资源"或"添加资源"

  5. 填写资源信息：
     - 资源名称: 用户特征数据
     - 资源描述: 用户基本信息和行为特征
     - 添加字段:
       * user_id (string) - 用户ID
       * age (int) - 年龄
       * gender (int) - 性别
       * city (string) - 城市
       * income (float) - 收入
       * education (int) - 学历
     - 数据行数: 10000
     - 数据列数: 6

  6. 保存资源

  7. 验证：
     python3 your_system_tool.py list

  3. 切换到 Network (网络) 标签

  4. 创建资源时，找到保存/创建的API请求

  5. 右键点击请求 → Copy → Copy as cURL

  6. 或者查看:

     - Request Method (POST/PUT等)
     - Request Headers (Content-Type等)
     - Request Payload (请求体数据)

  7. 将这些信息提供给我，格式如下：

     Request URL:
     http://100.64.0.23:30811/prod-api/data/xxx/saveResource

     Request Payload:
     {
       "resourceName": "测试资源",
       "resourceDesc": "描述",
       "resourceType": "1",
       ...
     }

  8. 我会根据实际格式更新脚本，让它可以批量创建资源
