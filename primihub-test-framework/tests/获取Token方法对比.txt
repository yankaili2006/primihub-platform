方法1: 从浏览器直接获取Token (★★★★★ 最推荐)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

这是最简单、最可靠的方法！

步骤：

1. 浏览器访问: http://172.20.0.12:8080

2. 登录（用户名: admin  密码: 123456）

3. 按 F12 打开开发者工具

4. 切换到 Network (网络) 标签

5. 刷新页面 (F5)

6. 点击任意API请求（如 getOrganList、getUserList 等）

7. 右侧找到 Headers → Request Headers → 复制 token 字段值

8. 将token保存：
   cd /home/primihub/primihub-platform/primihub-test-framework/tests
   python3 get_token.py save node0 "你的token"

优点：
  ✅ 最简单，只需要复制粘贴
  ✅ 最可靠，不会失败
  ✅ Token长期有效（24小时）
  ✅ 不需要处理RSA加密
